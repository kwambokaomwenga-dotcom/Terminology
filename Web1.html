Definition

Web 1.0, spanning roughly 1991 to 2004, was the first, "read-only" era of the World Wide Web, characterized by static HTML websites, one-way communication, and limited user interaction. Acting as a "digital library," it consisted of decentralized, informational sites where a few content creators provided information for a large, passive audience.

Characteritcis

  • Static Pages: Sites were built with static HTML, meaning content did not change in response to user input, often requiring manual updates.
  • Read-Only Content: Users were "consumers" who could browse information but not interact, comment, or create content.
  • Basic Structure: Websites were largely text-based with simple graphics and limited multimedia.
  • Minimal Interaction: Guestbooks or basic email forms were the extent of user interaction.
  • Slow Internet Speeds: Limited bandwidth affected site performance and interactivity.

Features

  • Fixed, pre-built web pages without real-time updates.
  • Essential text links for navigation between pages
  • Used for distributing static content efficiently.
  • Simple directory-like search engines
