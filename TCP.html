Transmission Control Protocol/Internet Protocol

It is a whole set of related protocols and tools that help computers to communicate with each other. Some protocols that are used include: SMTP-sending email messages, FTP- llows files to be easily copied to and from remote sites.It has 4 layers which are application, transport, internet and network access. Each of these layers are used for sending and receiving od data in the network. Its simple and more pratical than the seven layer OSI model.It serves as the core framework of modern Internet and networking systems.

Roles

Among the roles of TCP/IP include :

  • Breaking of data into packets
  • Managing flow control
  • Ensuring all packets arrive correctly at the right destination
  • Routing of packets to ensure they reach correct destination

Its main benefits are relaiblity-ensure data integrity, standardization- act as universal language for network communication and
