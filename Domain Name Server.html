Definition

Translation of human-readable domain names into IP addresses enables computers to communicate over the internet.Operates in a hierarchical and distributed manner while improving performance through caching.

Roles

  • Stores DNS records
  • Responds to queries from clients or other DNS servers
  • Operates within the hierarchical DNS structure
  • Can perform either recursive or authoritative functions
  • Enables communication between users and web servers by supplying necessary DNS data.

Its structure include:Root-the topmost level of the DNS hierarchy represented by a dot (.) at the end of a domain name, top-Level Domains (TLDs) - the level directly below the root that defines domain extensions includes extensions like .com, second-Level Domains - the main domain name registered by an organization, subdomains - extensions of the main domain used for organization and hostnames - identifies specific servers or devices within a domain.
