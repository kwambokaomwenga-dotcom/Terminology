Web Browser

The first web browser, World Wide Web (later renamed Nexus), was created by Tim Berners-Lee in 1990.Application software that allows users to access, search, and view information on the World Wide Web by communicating with web servers is known as a web browser. It enables smooth navigation of websites by retrieving and displaying online content in an interactive format.

A browsers works like this:The user enters a website URL into the browser.The browser contacts a DNS (Domain Name System) server to convert the domain name into an IP address.Using the IP address, the browser sends a request to the web server .The server responds with web content such as HTML, CSS, images and scripts.The browser processes and renders this content into a readable webpage.

Roles

  • Retrieving Information
  • Rendering and Interpretation
  • Navigation and Linking
  • User Interface Management
