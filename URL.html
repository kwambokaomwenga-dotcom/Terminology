URL

A URL (Uniform Resource Locator) is a formatted text string serving as a unique web address to locate resources like webpages, images, or documents on the internet.It functions as a digital address, allowing browsers to request and retrieve specific content from servers via protocols.

A URL has key components. they include:

  • Protocol/Scheme: Defines how the resource is accessed.
  • Domain Name: Identifies the specific server.
  • Path: Specifies the exact location or file on the server.
  • Top-Level Domain (TLD): Indicates the domain extension
  • Port Number: Specifies the server port
