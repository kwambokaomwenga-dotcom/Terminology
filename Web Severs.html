Web Server

A web server is a computer (hardware) and its software that stores web content and delivers it to users over the internet using protocols.Request: A user's browser sends an HTTP request.Processing: The server receives the request, finds the data, and processes it.Response: The server sends the requested web content back to the browser. Web servers can be categorized based on their functionality, usage, and implementation.

Roles

  • Allowing for the hosting of websites and applications.
  • Delivers HTML, images, and other static files
  • Provides authentication methods and request filtering to enhance safety.
  • Supports file transfers
