Terminology

Definition

Web applications rely on a wide range of specialized terminologies that are used to describe how the internet and web technologies function. Understanding these terminologies is essential, as they form the foundation of how web-based systems are designed, accessed, and managed.

These include the World Wide Web (WWW), Transmission Control Protocol/Internet Protocol (TCP/IP), Web Browsers, Web Servers, Uniform Resource Locators (URLs), Domain Name System (DNS), Hypertext Transfer Protocol (HTTP), Intranet, Extranet, Multitier Architecture, File Transfer Protocol (FTP), Hypertext Markup Language (HTML), as well as the evolution of the web through Web 1.0, Web 2.0, Web 3.0, and Web 4.0.

Each term is presented on its own dedicated page to promote clarity and ease of understanding.
