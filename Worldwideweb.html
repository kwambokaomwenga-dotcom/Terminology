World Wide Web

It is just one of the many services the internet provides. It is a wide area hypermedia information retrival initiative aimiming to give access to laarge array of documents. Invented in 1989 by Tim Berners-Lee, it uses HTTP to transfer data and HTML to create pages, acting as the primary information retrieval service for over 63% of the world's population

Characteristics

Its main characteristics are:

  • Wide-area - it spans across the whole globe
  • Hypermedia - it contains various types of media
  • Information retrival - it allow users to retrieve pages just by clicking hypertext links, or entering addresses
  • Universal access - it can be accessed seamlessly in many diffrent systems.

The core components it has is the URL-Universal Resource Allocator:A unique address for locating web resources., HTTP - (HyperText Transfer Protocol): The set of rules for transferring web data and HTML - (HyperText Markup Language): The publishing format used for documents.
